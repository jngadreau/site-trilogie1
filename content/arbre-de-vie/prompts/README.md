# Prompts / recettes (Arbre de Vie)

Fichiers d’aide pour alimenter le champ **`instruction`** (et parfois le contexte) des appels `POST /ai/generate-markdown`.

| Fichier | Usage |
|---------|--------|
| [hero-instruction.example.txt](./hero-instruction.example.txt) | Exemple de consigne pour un bloc **hero** de page d’accueil — à copier-coller ou adapter. |

Les **extraits livret** restent dans `oseunpasverstoi-jeux1/Arbre de vie/booklet/` ; un script d’import pourra les injecter automatiquement plus tard (voir [roadmap](../../docs/roadmap-taches.md)).
