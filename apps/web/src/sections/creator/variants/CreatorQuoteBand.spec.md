# CreatorQuoteBand

Bandeau **citation** : une phrase forte en markdown, signature et ligne de rôle — sans image.

## Props JSON

| Champ | Type | Notes |
|-------|------|--------|
| `quoteMarkdown` | string | Citation (markdown léger, *italique* autorisé) |
| `name` | string | Prénom nom |
| `roleLine` | string | Ex. « Créatrice — Ose Un Pas Vers Toi » |
