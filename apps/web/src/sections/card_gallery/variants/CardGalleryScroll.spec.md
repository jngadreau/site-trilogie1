# CardGalleryScroll

Bandeau horizontal défilant ; **même ratio carte** que la grille (~**672×1877**), largeur de colonne alignée sur le hero strip (~**118px**, `--dl-deck-card-scroll-w`).

## Choisir les cartes

Même règles que **CardGalleryGrid** : `cards` > `cardSlots` > `cardNumbers`, plus `deckCardsBasePath` optionnel.
