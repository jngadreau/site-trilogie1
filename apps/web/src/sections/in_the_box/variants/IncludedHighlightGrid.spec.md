# IncludedHighlightGrid

Section **contenu du jeu** : **grille 2 colonnes** de petits blocs (titre + markdown), pour mettre en avant plusieurs éléments (format, livret, usage) sans liste à cocher.

## Props JSON

| Champ | Type | Notes |
|-------|------|--------|
| `sectionTitle` | string | H2 |
| `introMarkdown` | string | optionnel |
| `highlights` | array | 2 à 6 `{ "title", "bodyMarkdown" }` |

**`media` : `[]`**
